## The environment required for the repository

### Hardware Equipment Required

| Equipment | Quantity |
| :--- | :--- |
| PICO Neo 3 Pro | 1 |
| OpenManipulator-X | 2 |
| Senseglove Nova | 2 |
| Desktop computer (Windows/Linux environment installed) | 2 |

### Software Environment Required

* **Programming Languages:** C# / C++ / Python (A normal programming environment)
* **Dependencies:** Library files provided by the relevant device manufacturers must be installed.
* **IDE Recommendation:** For reference, it is recommended to use JetBrains IDE products. (I am a loyal user of JetBrains :> )
